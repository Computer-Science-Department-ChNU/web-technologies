Оператори та операнди

Оператор – найменша автономна частина мови програмування, визначальна команда чи набір команд, наприклад: присвоєння, додавання, множення тощо.

Операнд – аргумент операції (дані, що обробляються операцією). 

        let result = operand1 + operand2 - operand3;
        =, -, + - оператори
        operand1, operand2, operand3 - операнди

Оператори та операнди

Бінарні оператори – оператори, які приймають два аргументи та повертають один результат. 

        let result = x + y;

Унарні оператори – оператори, які використовують один аргумент та повертають один результат. 

        x++;
