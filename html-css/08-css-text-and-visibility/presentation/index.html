  1. Спадкування в CSS

  3. Властивості видимості елементів на сторінці

Вступ

На лекції ви познайомитеся з різними правилами CSS для стилізації тексту, і як працює успадкування стилів для елементів нащадків. Як можна приховувати елементи розмітки на сторінці та у чому різниця між різними.

Вивчивши матеріал даного лекційного заняття, студент зможе:
  • Надавати стилі для тексту елемента на власний розсуд.
  • Розуміти, як працює спадковість стилів у html.
  • Сховати і робити прозоріші елементи.
  • Розрізняти у чому різниця роботи властивостей, які приховують елементи.

Спадкування в CSS

Спадкування – застосування стилів до елемента та його нащадків, які вкладені у ньому.

Більшість css властивостей неуспадковані як: width, height, border, background і т.д

Успадковані характеристики - це більшість тих правил, які впливають поведінка тексту: text-align, color, font, letter-spasing і т.д

Наприклад тег розташований усередині. Взявши фокус на елемент div-a, що проглядається, і через інспектор в панелі зі стилями можна знайти подібний запис (попередньо). Як видно властивість color успадковувалося, а background - ні

Властивості для тексту

  • font – універсальна властивість, що складається з властивостей, що визначають сімейство шрифту, розмір шрифту, зображення шрифту і т.д.
  • text-align – горизонтальне вирівнювання тексту в межах контентної частини елемента.
  • text-indent – відступ першого рядка у тексті.
  • letter-spacing – визначає відстань між символами елемента.
  • word-spacing – визначає відстань між словами елемента.
  • text-decoration - оформлення тексту лінією.
  • text-transform - перетворення (трансформація) тексту в великі або великі літери.
Список css властивостей для тексту вказано не весь!
