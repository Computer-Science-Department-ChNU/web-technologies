Дано одномірний масив.
    

    Його елементами можуть бути значення різних типів, включаючи: undefined, null, boolean, string, number."
    

    Наприклад:


const array = [5, undefined, 0, false, '', null, true, 1]


Масив може бути розрідженим (sparse array), тобто містити незаповнені елементи (empty slots).


Наприклад, додамо до масиву array елемент з індексом 15:


array[15] = 'новий елемент'
console.log(array)
// [ 5, undefined, 0, false, '', null, true, 1, <7 empty items>, 'новий елемент']


Напишіть функцію підрахунку незаповнених елементів масиву.
