Лекція 7. Робота з відступами, властивість margin

План лекції:

  1. Зовнішні відступи елемента
  2. Провалювання (схлопування)
  3. Поведінка при змішаному типі елемента

Властивість margin

margin – зовнішні відступи елемента, з усіх чотирьох сторін.

Також можна прописати лише конкретну сторону відступу:

  • margin-top – верхній відступ
  • margin-right – відступ справа
  • margin-bottom – нижній відступ
  • margin-left – відступ з лівої сторони

Кожну сторону можна задати за скороченим записом margin! Принцип за годинниковою стрілкою - { margin: 20px 30px 40px 50px; }

margin: auto;

Значення auto визначає автоматичний відступ обчислюваний браузером

Механіка margin

Значення auto за замовчуванням не працює вертикальною віссю.

Дочірній елемент відштовхується від батьківського верхнього та нижнього краю лише за наявності padding чи border властивості не зі значенням 0!

Відступи не підсумовуються по вертикалі.

По горизонталі відступи підсумовуються.

Резюме

  • Спадкування в CSS - можливість використовувати стилі від батьківського елемента в нащадку
  • Прихований елемент - елемент, що займає своє місце на сторінці і візуально ховається.
