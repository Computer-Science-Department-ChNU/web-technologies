Визначення розміру елемента

Властивості width і height не працюють правильно з рядковими елементами!

  • width – ширина елемента
  • height – висота елемента
  • padding – внутрішній відступ
  • border – межі (рамка) елемента

margin – зовнішній відступ елемента та не впливає на його розміри
