Умовні конструкції

Умовні конструкції чи оператори розгалуження - конструкції мови програмування, які забезпечують виконання певної команди (набору команд) лише за умов істинності деякого логічного висловлювання.

Оператор розгалуження if на основі виразу в дужках виконує чи пропускає блок коду. 

        let x = 10;
        let y = 20;
        if (x > y) {
          console.log('Значення x більше значення y')
        }

if/else

Оператор if може використовувати необов’язковий блок else, в якому буде знаходитися набір інструкцій, що запускаються, якщо умова хибна 

        let x = 10;
        let y = 20;
        if (x > y) {
          console.log('Значення x більше значення y')
        } else {
         console.log('Значення x менше значення y')
        }

Логічні операції

Логічні операції - операції над виразами, дозволяють отримати складний вираз, виходячи з простіших. 

        if (inputLogin == login && inputPassword == password) {
          console.log('Ви авторизовані!')
        }

Логічні операції

В JavaScript використовуються три логічні оператори:

  • Кон’юнкція - та (&&). Результат логічної операції true, якщо обидва аргументи true, інакше – false.
  • Диз’юнкція - або (||). Результат логічної операції true, якщо хоча б один із аргументів дорівнює true, інакше – false.
  • Заперечення - не (!). Приймає один аргумент та повертає протилежне значення.

Завдання

Який результат alerts:

        alert( null || 2 || undefined );
        alert( alert(1) || 2 || alert(3) );
        alert( 1 && null && 2 );
        alert( alert(1) && alert(2) );
        alert( null || 2 && 3 || 4 );

Напишіть умову if, щоб перевірити, чи значення age НЕ знаходиться між 14 та 90 включно.
