Спадкування

Більшість css властивостей не успадковані як: width, height, border, background і т.д.

Як видно властивість color успадковувалося, а background - ні Успадковані характеристики - це більшість тих правил, які впливають поведінка тексту: text-align, color, font, letter-spasing і т.д

Наприклад тег розташований усередині. Взявши фокус на елемент div-a, що проглядається, і через інспектор в панелі зі стилями можна знайти подібний запис (попередньо).

Властивості для тексту

  • font – універсальна властивість, що складається з властивостей, що визначають сімейство шрифту,
  • розмір шрифту, зображення шрифту і т.д.
  • text-align – горизонтальне вирівнювання тексту в межах контентної частини елемента.
  • text-indent – відступ першого рядка у тексті.
  • letter-spasing – визначає відстань між символами елемента.
  • word-spasing – визначає відстань між словами елемента.
  • text-decoration - оформлення тексту лінією.
  • text-transform - перетворення (трансформація) тексту в великі або великі літери.
Список css властивостей для тексту вказано не весь!
