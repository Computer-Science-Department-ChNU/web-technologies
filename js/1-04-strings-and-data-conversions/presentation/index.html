Що таке літерал, рядкові літерали

Літерал – явно вказане в коді значення. Їх особливість полягає в тому, що вони мають фіксовані значення і не можуть змінюватися під час виконання програми. Тобто, літерали використовуються для представлення значень у JavaScript.

        let x = 10;
        let - ключове слово
        x - ідентифікатор
        10 - літерал

Визначення рядкової змінної

Створити змінну типу string можна за допомогою рядкового літералу.
Рядковий літерал – послідовність символів, укладених в одинарні лапки ('') або подвійні лапки (""). 

        let firstName = "Ivan";
        let lastName = 'Romanov';

Конкатенація – операція я з'єднання (зчеплення) рядків. 

       let fullName = firstName + " " + lastName; // fullName = "Ivan Romanov"

Template literal

Template literal (шаблонні рядки) – використовуються для створення рядка на основі шаблону та значень, визначених у змінних, або є результатом виразів. Шаблонні рядки створюються з використанням зворотних лапок (``)

Шаблонний рядок

        let fullName = `${firstName} ${lastName}`;
        // fullName = "Ivan Romanov"
        let greeting = `Привіт, мене звати ${firstName}!`
        // greeting = "Привіт, мене звати Ivan!"

Перетворення типів

Перетворення типів – перетворення значення одного типу на значення іншого типу. Може відбуватись автоматично або за допомогою спеціальних функцій. 

        let message = 123; - як нам перевірити тип даних?
        let value = 1;
        let result;
        result = String(value);
        result = Number(value);
        result = Boolean(value);

Взаємодія: alert, prompt, confirm

alert - показує модальне вікно та чекає, доки користувач не натисне кнопку “ОК”.

prompt - Вона показує модальне вікно з текстовим повідомленням, полем, куди відвідувач може ввести текст, та кнопками ОК/Скасувати.

confirm - показує модальне вікно та двома кнопками: ОК та Скасувати. Результат: true, якщо натиснути кнопку OK, інакше — false. 

        let age = prompt('Скільки вам років?', 100);
        alert(`Вам ${age} років!`); // Вам 100 років!
        let isBoss = confirm("Ви бос?");
        alert(isBoss)
