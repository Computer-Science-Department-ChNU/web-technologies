Об’єкт

Об’єкт - сукупність пов’язаних даних та функціональних можливостей. Об’єкти містять властивості та методи.

Властивості – стан об’єкта, значення будь-якого типу.

Методи – поведінка об’єкта, функція, виконує будь-яку операцію. 

        let objectAsConstructor = new Object() - виклик конструктора
        let objectAsLiteral = {} - літерал об’єкта

Об’єкти

JavaScript об'єкт має властивості, асоційовані з ним. Властивість об'єкта можна розуміти як змінну, закріплену за об'єктом. Властивості об'єкта по суті є тими самими змінними JavaScript, за винятком, що вони закріплені за об'єктом. Властивості об'єкта визначають його характеристики.

Об’єкти

Об’єкти

Оскільки JavaScript - мова з динамічною типізацією, під час створення об'єкта не потрібно попередньо визначати тип даних для нього.

Він спочатку створюється порожнім і залежно від дій над ним виходять властивості та методи.

Після створення об'єкта йому можна додати властивість, для цього потрібно просто надати властивості значення. Якщо такої властивості немає – вона буде створена.

При спробі прочитати значення неіснуючої властивості буде отримано значення undefined.

Об’єкти

Найзручніший спосіб створення об'єкта – використання літералу {}. При цьому підході відразу можна вказати властивості та методи для об'єкта, що створюється.

Для отримання доступу до властивостей об'єкта найчастіше використовується оператор «.»: об’єкт.ім’я_властивості.

Звертатися до властивостей із рядковими іменами можна через квадратні дужки, як і до елементів масиву, наприклад, об'єкт[“ім'я_властивості”].

Використовуючи ключове слово delete, можна видаляти властивості з об'єкта.

В якості об'єкта може бути інший об'єкт. При цьому кількість вкладених об'єктів та рівень вкладеності не обмежується.

Об’єкти: Dot notation vs Bracket notation

  • Використовується, коли назва властивості є допустимим ідентифікатором (без пробілів, спецсимволів, не починається з цифри).
  • Коротший та читабельніший синтаксис.
                const user = {
                  name: "John",
                  age: 30
                };
                console.log(user.name); // "John"
                console.log(user.age);  // 30
  • Використовується, коли ім'я властивості є рядком, зберігається в змінній або містить недопустимі символи.
  • Дає змогу динамічно звертатися до властивостей.
                const user = {
                  "first name": "John",
                  age: 30
                  role: "admin"
                };
                // Доступ до властивостей з пробілами або спецсимволами
                console.log(user["first name"]); // "John"
                // Доступ через змінну
                const key = "role";
                console.log(user[key]); // "admin"
